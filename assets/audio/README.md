# Background Music

Para adicionar a trilha sonora de fundo:

1. Baixe uma música royalty-free (licença livre para uso)
2. Recomendações de fontes:
   - [Pixabay Music](https://pixabay.com/music/)
   - [Free Music Archive](https://freemusicarchive.org/)
   - [Incompetech](https://incompetech.com/music/royalty-free/)
   - [Bensound](https://www.bensound.com/)
3. Converta para MP3 se necessário
4. Renomeie o arquivo para `background_music.mp3`
5. Coloque o arquivo nesta pasta (`assets/audio/`)

O app tentará tocar automaticamente ao iniciar. Se o arquivo não existir, o app continuará funcionando normalmente, apenas sem música de fundo.
